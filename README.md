## 🔧 Technical Installation

```bash
# Clone the repository
git clone https://github.com/vram-ai/sigmai.git

# Install dependencies
pnpm install

# Start the development server
pnpm start

# Configuration
cp .env.example .env

Edit .env with your keys

# Start in development mode
pnpm dev
```
